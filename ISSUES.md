## Summary

Resumable 3-phase audit completed with active model id `github_copilot::gpt-5.5`. Phase 1 built a SLOC+tiktoken chunk plan: 35 counted source/config/doc files, ~17.2k SLOC, ~190k estimated tokens, packed into 4 review chunks targeting 64k tokens. Phase 2 reviewed the chunks for security and complexity risks across Python and Rust implementations. Phase 3 condensed findings to the highest-impact issues suitable for `ISSUES.md`.

The most critical risks are local command execution in non-interactive mode, network egress in “read-only” modes, SSRF gaps in `webfetch`, symlink-based workspace escapes, unsafe auto-writes, local provider credential leakage, and several supply-chain/secret-retention issues.
